# Trabalho de DevOps
Este repositório foi criado para a realização da atividade *hands_on5*, dada pelo docente *Carlos Alberto da Silva Junior*. Esse trabalho explora a utilização do GitHub e da organização em equipe, praticando conceitos como:
- Pull Request
- Branches
- Revisão de PRs
- Comunicação e Organização de Equipe

## 🎯 Objetivo e Finalidade
Além dos pontos de prática com versionamento e utilização do GitHub, esse trabalho também expõe curiosidades sobre os integrantes, com cada um incluindo um local que gostaria de conhecer um dia. Entre países lindos e cidades encantadoras, o brilho da evolução individual nesse trabalho foi o que se sobressaiu.
